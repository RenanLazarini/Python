Este código não utiliza dados reais e não envolve informações sensíveis.

Este script foi desenvolvido para a criação de uma base de dados fictícia de
satisfação de clientes, simulando um cenário real de vendas e avaliações de produtos.

A base gerada contém informações como ID da venda, produto, nota atribuída,
avaliação final (Good ou Bad), feedback associado e data da avaliação.

A lógica aplicada busca representar de forma coerente o comportamento esperado
em um cenário real, onde notas mais baixas resultam em avaliações negativas e
feedbacks condizentes, enquanto notas mais altas geram avaliações positivas.

O objetivo principal deste código é viabilizar testes, análises exploratórias e
criação de dashboards em ferramentas de BI, sem a necessidade de utilizar dados
reais ou confidenciais.

Ao final do processo, o script gera automaticamente um arquivo Excel com a base
completa, pronta para análise.
