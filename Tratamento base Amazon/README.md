Este código não utiliza dados sensíveis e foi aplicado sobre uma base pública.

Este script foi criado com o objetivo de realizar o tratamento e organização de
uma base de dados de produtos, preparando as informações para análise e uso em
ferramentas de BI.

A base original apresentava diversos desafios comuns em dados reais, como valores
monetários armazenados como texto, campos numéricos não padronizados, categorias
hierárquicas concentradas em uma única coluna e informações de produtos e clientes
misturadas.

O tratamento realizado inclui a limpeza e conversão de valores monetários, ajuste
de campos numéricos, separação de categorias e subcategorias e organização dos
dados em estruturas mais adequadas para análise.

Como resultado, os dados foram organizados em tabelas separadas de produtos e
clientes, facilitando a modelagem, o relacionamento entre tabelas e a construção
de dashboards no Power BI.

Ao final do processo, o arquivo tratado é exportado em formato Excel, pronto para
uso analítico.
